# v0.2.0 — Public archive

AgentHub Visual Manager v0.2.0 is the final public archive of the project.

Highlights:

- Chinese-first local Agent collaboration dashboard.
- Read-only `.agent-hub` observation and browser-side local import.
- Session-bound independent development for trusted local Git worktrees.
- Bounded text edits, fixed project validation and localhost browser acceptance.
- Memory-only Provider credentials and redacted durable evidence.
- Explicit cost ceilings, safe stop/resume and independent review gates.

Archive notes:

- Published as a sanitized single-commit snapshot.
- Original private Git history and internal development records are intentionally excluded.
- No ongoing maintenance, support or security-update commitment is provided.
- Bundled identifiers, paths, dates and hashes are synthetic placeholders.
- Unverified generated binary artwork is excluded; code-generated visual fallbacks remain.
