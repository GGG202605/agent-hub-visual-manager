# Security Policy

## Archived project

AgentHub Visual Manager v0.2.0 is an archived snapshot. It is not actively maintained and does not receive guaranteed security fixes.

## Safe use

- Run it only on a trusted local machine and only against repositories you trust.
- Keep the local service bound to localhost; do not expose it to an untrusted network.
- Use short-lived, limited Provider credentials where possible, and never commit credentials or local runtime records.
- Review generated changes and command results before making any Git commit or remote change.

## Vulnerabilities

Because the repository is archived, public issue reporting and response are not guaranteed. Do not post secrets or exploit details publicly. If you discover a vulnerability, stop using the affected capability and maintain a private fork with the required fix.
